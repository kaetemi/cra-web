Chroma Rotation Averaging in LAB color space. Rotates the AB chroma plane at multiple angles (0°, 30°, 60°), performs histogram matching at each rotation, then averages the results. This prevents color flips and preserves complex color relationships.

Pure Rust implementation compiled to WebAssembly. Fast and runs entirely in the browser.

Binned histogram matching with 256 bins. Classic approach with dithering for quantization.

Our method: Randomly selects between Floyd-Steinberg and JJN kernels per-pixel using lowbias32 hash, with standard scanning. Used for histogram processing quantization (when not using f32 histogram).

When enabled, uses joint colorspace-aware dithering for histogram quantization. Considers perceptual color distance when quantizing Lab values.

Our method: Randomly selects between Floyd-Steinberg and JJN kernels per-pixel using lowbias32 hash, with standard scanning. Used for final output quantization.

When enabled, uses joint RGB colorspace-aware dithering for final output. Processes all three RGB channels together using perceptual color distance.

Penalizes quantization choices that would cause the opposing point to fall outside the color gamut. Reduces color fringing artifacts.

This method is only available with WASM implementation. Please enable WASM or select a different method.
