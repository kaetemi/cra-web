<linear-img> Element

Drop-in image element with correct linear RGB resizing

How it works: The <linear-img> custom element displays images using proper linear RGB interpolation instead of the browser's sRGB scaling. This prevents the darkening and color shifts that occur with standard <img> tags when images are resized.

Usage

<script type="module" src="linear-img.js"></script>

<linear-img src="photo.jpg" width="200" height="150"></linear-img>
<linear-img src="photo.jpg" style="width: 50%;"></linear-img>
<linear-img src="photo.jpg" method="bilinear" fit="cover"></linear-img>
